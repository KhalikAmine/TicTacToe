## Getting Started
1. Navigate to the directory
   - cd C:\Users\Amine\eclipse-workspace\TicTacToe\src\TicTacToe
2. Compile program
    - javac Game.java
    - java Game

### Prerequisites

Before running the TicTacToe game, ensure you have the following installed:
